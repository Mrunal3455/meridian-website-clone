🍦 Meridian Ice Cream — Website Clone
A pixel-perfect clone of the Meridian Ice Cream website built using pure HTML5 and CSS3. Deployed live using GitHub Pages.
---
📌 Features
Responsive navigation bar with logo and links
Full-screen hero section with background image
Clean layout with sections for Flavours, About, and Contact
Styled using custom CSS — no frameworks used
---
🛠️ Tech Stack
HTML5 — structure and semantic markup
CSS3 — styling, layout, flexbox
